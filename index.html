O Cenário Atual

A operação hoje
vive em planilhas

A Vitrine Global gerencia toda a operação em planilhas Excel — 17 abas sem integração automática, gerando riscos e limitando o crescimento.

📊

17 abas de planilha

Cargas, itens, orçamentos, pedidos, financeiro, estoque — tudo separado e sem integração automática.

🔢

Cálculos manuais

Custo unitário BRL, rateio de frete, cotação USD por etapa — tudo calculado manualmente com risco de erro.

🏢

Sem visão multi-filial

Cada filial mantém seus próprios controles. Não há consolidação automática de resultados.

🛒

Marketplace manual

Pedidos do Mercado Livre, Shopee e Amazon controlados manualmente, sem sincronização de estoque.

09
Decisões Técnicas

Stack tecnológica escolhida

⚛️
Frontend

React 18 + TypeScript + Vite

Interface rápida, tipada e com hot-reload instantâneo.

🎨
UI

shadcn/ui + Tailwind CSS

Design system consistente e customizável.

⚡
Backend

Node.js + Fastify

Servidor de alta performance. 2× mais rápido que Express.

🔗
API

tRPC

Type-safety total entre front e back. Sem contratos manuais.

🐘
Banco de Dados

PostgreSQL 16 + Drizzle ORM

Arrays nativos, JSONB, colunas geradas e performance analítica.

🔐
Auth

JWT + bcrypt Local

Sem OAuth externo. Controle total sobre sessões.

📬
Filas

BullMQ + Redis

Sync de marketplaces assíncrono sem travar a UI.

📦
Monorepo

Turborepo

Tipos TypeScript compartilhados entre front e back.

🚀
Deploy

Railway → AWS ECS

Deploy simples no início, migração para AWS quando escalar.
